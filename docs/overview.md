## Content Manager
เมนูหลักที่ใช้ในการจัดการเนื้อหา 

สำหรับ User ระดับ Editor จะสามารถจัดการ
* Homepage
* Entry Popup
* Floating Popup
* Navigation
* Navigation Item
* Campaign & Condition
* Partner
* GWP
* FAQs
* Submenu

สำหรับ User ระดับ Admin จะสามารถจัดการเพิ่มเติมได้กับ
* Homepage Icon

ที่เหลือนอกจากนั้น(หากเห็น) จะเป็น collection สำหรับเซ็ทระบบของ dev เช่น content ratio, padding

### Publish/Unpublish

<img src="/assets/images/publish-unpublish-date.png" alt="Publish/Unpublish" style="width:100%; max-width:200px; hight:auto;">
